Major Functionalities and Controls:

​There are options to play : Keyboard or Mouse

Keyboard controls: ‘​w’ to move up

‘a’ to move left

‘s’ to move down

‘d’ to move right

‘z’ and ‘x’ for cheat records

Mouse controls: ​Move the Player using mouse.
